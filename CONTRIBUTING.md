# Đóng góp cho ChatbotTravel_client

Cảm ơn bạn đã quan tâm đến việc đóng góp cho dự án này!

## Cách đóng góp

1. Fork repository này và tạo nhánh mới từ `master`.
2. Thực hiện các thay đổi mong muốn.
3. Đảm bảo code tuân thủ các quy tắc định dạng và đã được kiểm tra.
4. Gửi pull request mô tả rõ ràng thay đổi của bạn.

## Quy tắc đóng góp
- Viết commit message rõ ràng, ngắn gọn.
- Đảm bảo không có lỗi build hoặc lỗi lint.
- Thảo luận trước khi thực hiện thay đổi lớn.

## Liên hệ
Nếu có câu hỏi, vui lòng mở issue hoặc liên hệ với maintainer của dự án.
